Line of sight

Line of sight analysis is used to identify points on the terrain surface along a given line that are visible or invisible from a given point on this line.

LoS or ray tracing is used for:

  • visibility analysis
  • emitor-receptor systems design taking into account decay of signal with distance, for example, to optimize locations of cell towers

Computing Line of sight

Given a viewing point A and a target B, we define $$ tan \alpha = (z_B-z_A )/ d_{BA} $$ B is not visible from A if,

  • for any point C between AB, $\alpha_c$ is greater than $\alpha$
  • if any point C is above the line that connects AB

Computing Line of sight

Given a viewing point A and a target B, we define $$ tan \alpha = (z_B-z_A )/ d_{BA} $$ B is visible from A if,

  • for any point C between AB, $\alpha_c$ is smaller than $\alpha$
  • if any point C is below the line that connects AB

Viewshed

  • region defined by a set of grid cells (or triangles) visible from a given viewing point
  • computed by applying LOS analysis from the viewing point to each cell within a given distance
  • can be implemented for TIN or raster DEMs

  • binary visibility map (yes/no)
  • angle above/below horizontal plane
  • relative elevation above/under horizontal plane
  • what is visible from the given point
  • from where one can see the given point

Cumulative Viewshed

Cumulative viewshed: set of grid cells visible from a set of points (e.g. along a road) applications find a path that has minimum or maximum visibility optimize distribution of signal sources to achieve the maximum coverage limit development to areas that are not visible - e.g. from the Blue Ridge Parkway minimize visibility of wind turbines
